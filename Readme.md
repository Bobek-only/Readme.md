<i>We recommend using</i>yarn for installing packages, but you can use npm instead;

$ npm install

3.Create first build

$ yarn run build:prod

4.Copy .evn.example file to.evn and make the necessary changes there

5.Run project in Dev mode

$ yarn run dev
